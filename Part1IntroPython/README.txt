Files & Folders Description:

IntroductionToPythonNotes.pdf -- pdf slides

examples -- folder containing code examples. The vast majority of examples are presented as Jupyter notebook files (.ipynb)
	- name of each Jupyter notebook files starts with 's#', which matches the slide number in the top left corner of the slide on which that example is mentioned
	- 'python_files' folder contains examples of Python files that are mentioned in the notes. There are not that many of them and they are not numbered
	- 'data_files' folder contains data that is used by some of the .ipynb and .py files.
	- 'output_files' folder is currently empty, but this is where any output from example files will be saved to
	- 'misc_examples' folder containing more examples (these were not matched to a particular file)
